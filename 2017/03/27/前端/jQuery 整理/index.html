注意点：

  • 样式的优先级：外部样式 < 内部样式 < 内联样式
    .addClass()方法是通过增加class名的方式，那么这个样式是在外部文件或者内部样式中先定义好的，等到需要的时候在附加到元素上
    通过.css()方法处理的是内联样式，直接通过元素的style属性附加到元素上的
